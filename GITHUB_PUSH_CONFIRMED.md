### Quick Import Steps

1. **Open New Kiro Account**
2. **Clone Repository**
   ```bash
   git clone https://github.com/Brianchisolm10/The-Alliance-Website.git
   cd The-Alliance-Website
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Setup Environment**
   ```bash
   cp .env.example .env
   # Edit .env with your values
   ```

5. **Setup Database**
   ```bash
   npx prisma migrate dev
   npx prisma db seed
   ```

6. **Start Development**
   ```bash
   npm run dev
   ```

### Syncing Between Accounts

**Push changes from Account A:**
```bash
git add .
git commit -m "Description of changes"
git push origin main
```

**Pull changes in Account B:**
```bash
git pull origin main
npm install  # if package.json changed
npx prisma migrate dev  # if schema changed
```
